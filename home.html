Calculate the maximum string size for a photovoltaic installation. The method is consistent with the NEC 2017 690.7 standard.

Module



Provide the location of installation and details on module and installation type. 


  • Weather: Weather data is obtained by default from the PVgis data source.It is possible to load your own file ('Load my weather') if you meet the following requirements:

    • CSV: must load a csv file delimited by commas (;). 

    • Data: the file must contain the following columns with the specified name.

      • Time: yyyy-mm-dd HH:MM

      • Year: yyyy

      • Month: mm

      • Day: dd

      • Hour: HH

      • Minute: MM

      • DNI: direct normal irradiance (W/m2)

      • GHI: global horizontal irradiance (W/m2)

      • DIF: diffuse horizontal irradiance (W/m2)

      • T2m: temperature (degree Celsius)

      • WS10m: 10-m total wind speed (m/s)

    • Time: you must specify the following statistics 

      • Hourly interval: interval between records (in hours).

      • Yearly interval: years contained in the dataset (in years).

Results



Using the weather data and the module parameters, the open circuit voltage is simulated using the single diode model. The table shows the recommended Voc values, voltage in Volts, cell temperature in Celsius and plane-of-array (POA) irradiance in W/m^2.


  • 690.7(A)(3)-P99.5 is the 99.5 percentile Voc value over the simulation time. This is the recommended value of Voc to use for designing PV string lengths. Statistically Voc will exceed this value only 0.5% of the year. Suppose that open circuit conditions occur 1% of the time randomly. In this case the probability that the weather and system design maintain the system voltage under the standard limit is 99.995%, i.e. max system voltage would statistically be exceeded for 26 minutes per year.


  • 690.7(A)(3)-P100 is the historical maximum Voc over the years of simulation.


  • 690.7(A)(1) is the traditional value used for maximum Voc. This is found using the mean minimum yearly dry bulb temperature (i.e. find the minimum temperature in each year and take the mean of those values) from the database. The Voc is calculated assuming 1 sun irradiance ( 1000 W/m^2), the mean minimum yearly dry bulb temperature and the module paramaeters.


  • 690.7(A)(3)-DAY is similar to the traditional value value, except the mean minimum yearly daytime dry bulb temperature is used as the cell temperature for calculating Voc. Daytime is defined as GHI greater than 150 W/m^2. The Voc is calculated assuming 1 sun irradiance (1000 W/m^2), the mean minimum yearly daytime dry bulb temperature and the module paramaeters

Figure 1 shows a histogram of open circuit voltage (Voc) values, with the y axis scaled to the number of hours per year spent at this open circuit voltage Note that this voltage value would only be reached for a system in open circuit conditions. Typically PV arrays operate at the maximum power point, with a voltage approximately 20% lower than the open circuit voltage.
